## 프로젝트 개요
:movie_camera:
영화 예매 & 리뷰 웹서비스<br/>

영화를 예매할 수 있고 감상한 영화의 평점과 리뷰를 남길 수 있는 웹 서비스


## 목적
- 프로젝트 생성부터 마지막까지 모든 과정을 경험
- Redux 사용 경험
- router라이브러리를 사용하지 않고 싱글페이지 개발
